# Tyche Contact WordPress Plugin
Tyche is a WordPress Plugin that will create a new contact in Mailjet when an order is placed in WooCommerce

It performs the following tasks
- Creates a new Contact on MailJet
- Sends an Email to the Contact along with First Name and Last Name
- Lists all Contacts created on MailJet 

## Installation

Tyche requires WordPress 

- Copy the Tyche folder into your plugin folder
- Activate the Plugin
- GO to the Plugin Settings page and put in your MailJet API Keys

Place an Order on your WooCommerce store and viola! A contact is created on MailJet and an Email is sent to the Client.

You can view the list of created contacts by viewing the Plugin Settings Page
